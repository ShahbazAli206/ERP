Import Business Management System (Demo / MVP Development Specification)
Objective

Develop a professional demo version of the Import Business Management System.

The goal is to demonstrate complete workflows, navigation, UI, dashboards, reporting, and module structure to the client.

This is NOT the final production implementation.

For this phase:

Use mock data.
Use local database.
Use fake services instead of real APIs.
Build clean architecture that can later integrate real APIs without changing UI.

Everything should be modular and production-ready.
